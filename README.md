# Handwritten Digit Recognition using CNN

## 📌 Project Overview
This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits from 0–9 using the MNIST dataset.

## 🧠 Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## 📂 Dataset
MNIST dataset:
- 70,000 images
- 28x28 grayscale images
- Digits from 0 to 9

## ⚙ Model Architecture
- Convolution Layer
- ReLU Activation
- Pooling Layer
- Flatten Layer
- Dense Layer
- Softmax Output Layer

## 📊 Results
Achieved approximately 98% accuracy on test data.

## 🎯 Conclusion
This project demonstrates the power of CNN in image classification tasks.
